## 2. Gestión (Cronograma y Análisis de Costos)

Esta sección detalla cómo se administraron los tiempos y los recursos económicos del proyecto, diferenciando claramente entre la construcción física que se completó y la parte tecnológica que quedó como una propuesta técnica.

### Cronograma: Avances y Desafíos en la Construcción

A diferencia de lo planeado inicialmente, el proyecto logró completar etapas físicas fundamentales. La fabricación no se detuvo por falta de materiales, sino por la complejidad encontrada durante el ensamblaje de los componentes impresos.

* **Armazón Estructural (Finalizado):** Se completó satisfactoriamente la creación del chasis interno de la prenda. Para esto, se utilizó alambre estructural y cinta de aislar, la cual fue esencial para asegurar todas las uniones y dar forma a la silueta de la armadura.
* **Compra de Materiales (Finalizada):** Se adquirió el total de los materiales necesarios para la parte física de la prenda, incluyendo todo el plástico para impresión (filamento), el metal para el armazón y el recubrimiento textil.
* **Proceso de Impresión (Ejecutado):** Se utilizó el filamento adquirido para producir las piezas de polímero que conforman la cobertura externa.
* **Ensamblaje y Electrónica (Inconclusos):** Aunque el armazón está terminado, no se llegó a cubrir con el textil (vinipiel). De igual forma, los componentes electrónicos no se compraron, ya que el proyecto se centró en la validación de la estructura y el mecanismo de las escamas.

| Etapa del Proyecto | Estado Final | Observaciones |
| :--- | :--- | :--- |
| **Modelado 3D** | Completado | Se diseñaron todas las piezas de las escamas y el sistema de tracción. |
| **Adquisición de Material Físico** | Completado | Se compró todo el filamento, alambre, vinipiel y cinta necesarios. |
| **Construcción del Armazón** | Completado | La estructura de alambre y cinta de aislar está terminada al 100%. |
| **Producción de Piezas 3D** | Completado | Se imprimieron las piezas utilizando el material adquirido. |
| **Compra de Electrónica** | No ejecutado | Estos materiales se mantuvieron únicamente como una proyección técnica. |
